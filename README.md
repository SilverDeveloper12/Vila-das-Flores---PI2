# Projeto "Vila das Flores" criado com o intuito de realizar um web-site utilizando a framework "Angular".
#
#
  O projeto Vila das Flores foi realizado pelo grupo 3 da Turma - A do segundo semestre no curso "Tecnologia em Análise e Desenvolvimento de Sistemas" na instituição
  SENAC Nações Unidas, para a matéria "Projeto Integrador: Desenvolvimento Estruturado de Sistemas" lecionada pelo professor Wilson Lourenço.
#
 A matéria "Projeto Integrador: Desenvolvimento Estruturado de Sistemas" tem o intuito de instigar os alunos do curso a trabalhar em equipe e desenvolver um web-site
 utilizando a framework de HTML "Angular", que cria interfaces dinâmicas utilizando a técnologia recente e padronizada para criação de web-sites, a ferramenta
 Angular é mantido pela empresa "Google" que dá apoio aos usuários de sua ferramenta com atualizações e mudanças contínuas para a melhor experiência de mercado.
#
 A ferramenta Angular é open-source, portanto os desenvolvedores podem também alter o código fonte da ferramenta e criar suas prórprias soluções, Angular também é focada
 na parte "Client-Side" o que significa que o foco é na aparência do web-site e na experiência do cliente e a interface dinâmica do web-site.
